# Keypad-Based-Security-System
Arduino based Keypad and LCD Interface System using 4x4 Matrix Keypad and 16x2 LCD.

## Project Overview

This project demonstrates interfacing a 4x4 Matrix Keypad with a 16x2 LCD Display using Arduino UNO.

## Components Used

- Arduino UNO
- 4x4 Matrix Keypad
- 16x2 LCD Display
- Potentiometer
- Breadboard
- Jumper Wires

## Features

- Keypad input detection
- Real-time LCD display output
- User-friendly interface
- Embedded C programming

## Technologies Used

- Arduino IDE
- Embedded C
- Tinkercad
